per cambiare l'unità di misura bisogna mettere dopo la stringa un underscore ( _ ) seguito dall'unità di misura e chiudere il tutto con il punto e virgola.Se non viene specificata nessuna unità di misura userà quelle default di ue quindi cm cm/s e gradi.

esempio:

Telemetry VehicleSimData.ChassisSimData.Speed;VehicleSimData.ChassisSimData.Speed_m/s;VehicleSimData.ChassisSimData.Speed_km/h;
